
6. Clases y Métodos const

Sirven para indicar que algo no debe cambiar.

class Persona {
private:
    string nombre;

public:
    Persona(string n) : nombre(n) {}

    string getNombre() const {
        return nombre;
    }
};

----

7. Inicialización con lista de miembros

Una forma más elegante de inicializar atributos:

class Punto {
private:
    int x, y;

public:
    Punto(int _x, int _y) : x(_x), y(_y) {}
};
